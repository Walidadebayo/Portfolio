Projects

Welcome to my showcase of projects! I am excited to present to you a collection of innovative and impactful attempt that I have passionately worked on, utilizing my skills and expertise.

MappifySQL

MappifySQL is a lightweight, easy-to-use Object-Relational Mapping (ORM) library for MySQL databases, designed for use with Node.js. It provides an intuitive, promise-based API for interacting with your MySQL database using JavaScript or TypeScript.

AdealTutor

AdealTutor is an educational website currently under development. This website is built with HTML, CSS, Bootstrap, JavaScript, Jquery, NodeJS, ExpressJS, MySQL, and EJS. This website will allow students to register, login, view and download past papers and revision notes for different qualification, exam boards and subjects.

AdealTube

AdealTube is a YouTube video downloader website built with HTML, CSS, Bootstrap, JavaScript, Jquery, NodeJS, ExpressJS and EJS. This website allow users to quickly and conveniently download videos from YouTube in MP4, MP3, and numerous other formats by performing a search or pasting YouTube video URL / link in the textbox on the website.

Alusoft Mall

Alusoft Mall is an e-commerce website built with HTML, CSS, Bootstrap, JavaScript, Jquery, Ajax, NodeJS, ExpressJS, MySQL and EJS. This website enable users to register, login, view products details, search for products, view product by category, add product to cart, add to wishlist, checkout products and many more.

Nasltes

Nasltes is a ticket booking system built with HTML, CSS, Bootstrap, JavaScript, Jquery, NodeJS, ExpressJS, MySQL, and EJS. This website is made for department of science laboratory technicians at LAUTECH University to enable them to book tickets for their events. This website allow students to book their tickets and download their tickets from the confirmation message sent to their email address after booking their ticket in PNG format and also allow admin to manage the website.

BG Removal

BG Removal is a background image remover built with HTML, CSS, Bootstrap, JavaScript, Jquery, NodeJS, ExpressJS, and EJS. This website allow users upload images from their device and remove background from images and download the image in PNG format. This was a team project and I was responsible for the back-end of the website. I used NodeJS, ExpressJS, and EJS to build the back-end of the website. I also used RemoveBG API to remove background from images.

ArtVibrance

ArtVibrance is website that showcase art works built with HTML, CSS, Bootstrap and JavaScript. This website generates random art works from Unsplash API and display them on the website.

Adeal Calculator

Adeal Calculator is a scientific calculator built with HTML, CSS, and JavaScript. This calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It can also perform scientific operations such as square root, square, cube, cube root, factorial, sin, cos, tan, log, ln, and many more.
