Projects

Welcome to my showcase of projects! I am excited to present to you a collection of innovative and impactful attempt that I have passionately worked on, utilizing my skills and expertise.

VizXpress

VizXpress is a powerful and user-friendly web application designed to simplify your image, gif and video processing needs. Built with Next.js, React.js, TypeScript, Tailwind CSS, and Flowbite React, it offers a suite of tools to help you achieve your goals effortlessly. The background remover API is developed using Python, Flask, and RemBG package. It's under development and more features will be added soon.

Predictif Finance

Predict.if Finance is a comprehensive financial analysis platform for professional investors, built with React.js, TypeScript, HTML, CSS, and Tailwind CSS. It utilizes React Router for seamless navigation and leverages AI models to analyze vast streams of news articles, alternative data, and traditional financial data, providing actionable insights through an intuitive and responsive user interface. This project was developed as a collaborative team effort and it's still under development.

MappifySQL

MappifySQL is a lightweight, easy-to-use Object-Relational Mapping (ORM) library for MySQL databases, designed for use with Node.js. It provides an intuitive, promise-based API for interacting with your MySQL database using JavaScript or TypeScript.

AdealTutor

AdealTutor is an educational platform providing students with access to past exam papers, revision notes, and AI-powered tools for creating flashcards, quizzes, and personalized notes.
Utilised technologies including HTML, CSS, Bootstrap, TailwindCSS, jQuery, Ajax, NodeJS, ExpressJS, EJS, MySQL, MappifySQL , and PassportJS.

Alusoft Mall

Alusoft Mall is an e-commerce website built with HTML, CSS, Bootstrap, JavaScript, Jquery, Ajax, NodeJS, ExpressJS, MySQL and EJS. This website enable users to register, login, view products details, search for products, view product by category, add product to cart, add to wishlist, checkout products and many more.

Nasltes

Nasltes is a ticket booking system built with HTML, CSS, Bootstrap, JavaScript, Jquery, NodeJS, ExpressJS, MySQL, and EJS. This website is made for department of science laboratory technicians at LAUTECH University to enable them to book tickets for their events. This website allow students to book their tickets and download their tickets from the confirmation message sent to their email address after booking their ticket in PNG format and also allow admin to manage the website.

ArtVibrance

ArtVibrance is website that showcase art works built with HTML, CSS, Bootstrap and JavaScript. This website generates random art works from Unsplash API and display them on the website.
